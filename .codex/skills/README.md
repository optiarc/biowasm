# Codex Skills

This repo uses Codex skills for common workflows.

## Available skills
- `commit-push`: stage all changes, create a single commit, and push.
- `commit-push-split`: split changes into multiple commits, then push.

## Usage
Mention a skill explicitly in your request, e.g.:
- "Use `commit-push` to commit and push"
- "Use `commit-push-split` to create 3 commits and push"
